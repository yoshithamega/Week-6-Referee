## Problem Statement
Choosing the right AWS compute service is difficult because each option has different trade-offs in cost, scalability, and management.

## Solution
This project builds a simple comparison tool that helps users choose between AWS EC2, Lambda, and ECS by explaining pros, cons, and trade-offs.

## How the Tool Works
1. The user enters budget preference
2. The user enters scalability requirement
3. The user enters management preference
4. The tool compares multiple services and explains trade-offs
